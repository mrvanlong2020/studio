Hãy xây cho tôi một web app quản lý vận hành studio chụp ảnh chuyên nghiệp bằng tiếng Việt, giao diện tối giản, dễ nhìn, dùng được trên điện thoại và máy tính. App này dành cho studio chụp cưới, beauty, fashion, gia đình và sự kiện.

MỤC TIÊU APP
Tôi cần một hệ thống quản lý toàn bộ quy trình của studio từ khách hàng -> báo giá -> tạo đơn -> thu tiền -> ký hợp đồng -> giao việc -> theo dõi chi phí -> tính lương nhân viên -> thống kê doanh thu/lợi nhuận -> báo cáo tháng.

YÊU CẦU QUAN TRỌNG
1. Bảng tạo đơn hàng và bảng thu tiền phải nằm CHUNG trong cùng 1 màn hình quản lý đơn hàng, không tách rời.
2. Trong hồ sơ khách hàng phải có:
   - Avatar
   - Họ tên
   - Số điện thoại
   - Facebook link
   - Ghi chú
   - Nguồn khách (Facebook, TikTok, giới thiệu, khách cũ, khác)
3. Sau khi báo giá xong phải có nút tạo hợp đồng ngay từ báo giá hoặc đơn hàng.
4. Phải có dashboard biểu đồ doanh thu, chi phí, lợi nhuận theo tháng.
5. Phải có thống kê công nợ khách hàng.
6. Phải có phân quyền user: Chủ studio / Quản lý / Sale / Kế toán / Photographer / Editor / Makeup.
7. Thiết kế app theo kiểu quản trị thật sự để dùng lâu dài, không làm demo sơ sài.

CÔNG NGHỆ
- Frontend: React + TypeScript + Tailwind
- Backend: Node.js + Express hoặc Next.js API routes
- Database: PostgreSQL hoặc Supabase
- Auth: đăng nhập bằng email + mật khẩu
- Charts: dùng thư viện biểu đồ đẹp, dễ đọc
- Hỗ trợ upload hình avatar khách hàng và file hợp đồng PDF
- Giao diện responsive, ưu tiên mobile first

Q. LOGIC TÍNH TOÁN QUAN TRỌNG
- Doanh thu = tổng tiền đơn hàng đã xác nhận
- Thu thực tế = tổng các khoản payment đã nhận
- Còn nợ = tổng tiền đơn - tổng payment
- Lợi nhuận gộp show = doanh thu show - chi phí show
- Lợi nhuận ròng tháng = tổng thu - tổng chi show - tổng chi vận hành - lương
- Tự động cập nhật số liệu dashboard khi thêm/sửa/xoá giao dịch
